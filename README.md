# 🚀 Snap Class Landing Page

A modern and responsive landing page built for the Snap Class AI Attendance Management System.

The landing page is designed to showcase the platform, explain its features, and provide a clean user experience for students, teachers, and visitors.

---

## 🌐 Live Demo

🔗 https://snapclass-landing-page-theta.vercel.app

---

## 📌 Overview

Snap Class Landing Page serves as the public-facing website for the Snap Class platform.

It introduces users to the system, highlights the key features, and provides information about the AI-powered attendance solution.

The project focuses on responsive design, modern UI/UX, and seamless accessibility across devices.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Bhanuprakashdev-prog/snapclass-landing-page.git
```

### Navigate to Project Directory

```bash
cd snapclass-landing-page
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
